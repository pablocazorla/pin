Float and Clear

Use these classes to set float and clear CSS properties to any tag.

CSS Property Use the class 
element{
	float: none;
}
.float-none 
element{
	float: left;
}
.float-left 
element{
	float: right;
}
.float-right 
element{
	clear: none;
}
.clear or .clear-none 
element{
	clear: left;
}
.clear or .clear-left 
element{
	clear: right;
}
.clear or .clear-right 
.clearfix {
	zoom: 1;
}
.clearfix:before,
.clearfix:after {
	content: '';
	display: table;
}
.clearfix:after {
	clear: both;
}
.clearfix (to avoid float collapses).

Position

Use these classes to set position CSS property to any tag.

CSS Property Use the class 
element{
	position: static;
}
.position-static 
element{
	position: relative;
}
.position-relative 
element{
	position: absolute;
}
.position-absolute 
element{
	position: fixed;
}
.position-fixed

Overflow

Use these classes to set overflow CSS property to any tag.

CSS Property Use the class 
element{
	overflow: visible;
}
.overflow-visible 
element{
	overflow: hidden;
}
.overflow-hidden 
element{
	overflow: scroll;
}
.overflow-scroll 
element{
	overflow: auto;
}
.overflow-auto

For overflow-x and overflow-y:

To set these properties, just use the -x or -y terms. Eg: .overflow-x-hidden, .overflow-y-scroll, etc.

Box sizing

Use these classes to set box-sizing CSS property to any tag.

CSS Property Use the class 
element{
  -webkit-box-sizing: content-box;
  -moz-box-sizing: content-box;
  box-sizing: content-box;
}
.box-sizing-content or .box-sizing-content-box 
element{
  -webkit-box-sizing: border-box;
  -moz-box-sizing: border-box;
  box-sizing: border-box;
}
.box-sizing-border or .box-sizing-border-box
