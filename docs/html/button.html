Button colors

All classes in Pin are 'unstyled', es decir que se ven como una serie de líneas de texto.

Button color Class Used mostly for
Button primary btn btn-primary Primary actions (eg: send, follow, acept, etc.).
Button secondary btn btn-secondary Secondary actions (eg: save, edit, etc.).
Button grey btn btn-grey Neutral actions (eg:cancel, close, etc.).
Button dark btn btn-dark Just a dark button.
Button link btn btn-link Render with link styles
Button danger btn btn-danger Danger actions (eg: delete)
Button warning btn btn-warning Warning actions (eg: trash)
Button success btn btn-success Success actions (eg: save, ok, exit).

Button shapes

All classes in Pin are 'unstyled', es decir que se ven como una serie de líneas de texto.

Button shape Class Description
Button btn btn-primary Default button, with round corners (4px).
Button flat btn btn-primary flat Button shape without rounded corners (flat style).
Button rounded btn btn-primary rounded With completely rounded corners.
+ btn btn-primary big circular Useful for icons.

Button Shape wrappers

An useful option to change all buttons from a section of the document, is by wrap all buttons in a tag with the classes .flat-buttons, .rounded-buttons or .circular-buttons, according to what you want.

Example 1 - All buttons flat, inside a tag with class .flat-buttons.

Button 1 Button 2 Button 3 Button 4 (rounded) Button 5

Note the Button 4. It have the class .rounded, so the button has rendered as a rounded button, establishing a rule exception for all flat buttons.

Example 2 - All buttons rounded, inside a tag with class .rounded-buttons.
