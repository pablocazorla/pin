Dropdown

Use dropdown to create custom menu, or selector, etc.

All elements to show have to be included inside the .dropdown-content tag.

You can fill the .dropdown-content tag with whatever you want.

Dropdown by click or roll over

By default the dropdown is open by click in the button. But you can open the dropdown by roll over the button adding the class .hover.

Example - Dropdown by click and dropdown by roll over.

You are clicked the button.
You are roll over the button.

Important:

- Is not necessary to set a .btn class for the .dropdown-trigger, but is important for the click dropdown to set an <a> or <button> tag.

- If you use a <a> as dropdown trigger, it's important to set the attribute href="javascript:void(0);" to avoid link navigation.

The .hover dropdown is useful for menus.
