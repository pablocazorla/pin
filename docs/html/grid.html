Grid

You can create a responsive an fluid grid layout.

Basic tag structure

To create a grid layout, you have to follow a classic structure, use in another CSS frameworks (eg: Bootstrap).

This structure is: a tag with class .row and a serie of tags with class .col: 

<div class="row">
  <div class="col">...</div>
  <div class="col">...</div>
  <div class="col">...</div>
  <div class="col">...</div>
</div>

Simple grid

In most cases (I think, in the 99.99% of cases) we use equal-width columns for our grids.

For these cases just use the classes .grid-#, where # is the number of columns per row.

Add a class .grid-# un the row tag, or in his parent (it's the same).

Gap options

We call gap to teh separation between columns. By default the gap is 3 rem width (1 rem = 10px, i.e. 3 rem = 30px).

But you can change this gap width using the right class .gap-# (# is the number of rem width).

This option is from 0 (no gap) to 9 (from .gap-0 to .gap-9).

Custom grid, with variable-width column

Sometimes, you could need to create a grid with columns of different width.

In these cases, you can use the .col-#part-#total (eg: .col-1-3, .col-3-4, etc.).

The #part refers the number of units over #total, the total number of units. For example, .col-1-3 refers to 1 part from the total of 3 parts, i.e. one third. The sentence also equals to the fraction 1/3 (33.3333%). Thus, the column with .col-1-3 class will have a width of 33.3333%.

Another examples: .col-2-3 = 66.6666%, .col-4-5 = 80%, .col-1-4 = 25%.

Offsetting columns

You can move the columns to the right using .offset-#part-#total classes. The numbers, of course, follow the same rule of size as the column width.

Example - Grid with a variety of offsets.

.col-2-3 .offset-1-3
.col-2-5 .offset-1-5
.col-2-5
.col-1-6 .offset-5-6
.col-1-3 .offset-1-3

Responsive grid options

All options for the grid layout could be set according the screen/device sizes by adding the terms md- (for medium devices like tablets) and sm- (for small devices like phones).

Term Examples Screen width Used for
md- .grid-md-3
.gap-md-5
.col-md-1-3
.offset-md-1-4 		990px Medium devices (tablets in landscape position).
sm- .grid-sm-3
.gap-sm-5
.col-sm-1-3
.offset-sm-1-4 		640px Small devices (phones).
